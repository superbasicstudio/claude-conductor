What Claude Conductor Does

Claude Conductor is a command-line tool that:

  • Creates markdown documentation files in your local project directory
  • Reads your local codebase to analyze structure (only when you run it)
  • All operations are performed locally on your machine
  • No data ever leaves your computer

Open Source Transparency

Our complete source code is available at github.com/superbasicstudio/claude-conductor. You can verify our privacy claims by reviewing the code yourself.

Local Storage and Cookies

Claude Conductor's documentation website uses local storage (not cookies) for the following purposes:

  • Theme Preference: We store your light/dark mode preference in your browser's local storage to maintain your preferred viewing experience across visits
  • No Tracking: This preference is stored only in your browser and is never transmitted to any server
  • No Third-Party Cookies: We do not use any tracking cookies or third-party analytics

You can clear this stored preference at any time by clearing your browser's local storage for this site.
